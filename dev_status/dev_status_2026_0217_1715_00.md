# 開発記録: グルーピングロジックの原状復帰（スキル別グルーピングの再導入）

## 日時
2026-02-17

## 内容
- `rys/group_requests.py` において、全実行可能トピックを一括して LLM に投げていたロジックを、スキルごとに分けて投げる以前の方式に戻した。
- 異なるスキルが混在した状態で LLM に投げると、論理的な境界の判定精度が低下し、リクエストが過剰に分割される問題が発生していたため。
- これにより、`role_grouper.md` の指示（同一スキル内での分析）と実装が一致し、関連するトピックが正しく集約されるようになる。

## 変更ファイル
- `rys/group_requests.py`
